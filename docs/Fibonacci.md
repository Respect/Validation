# Fibonacci

- `Fibonacci()`

Validates whether the input follows the Fibonacci integer sequence.

```php
v::fibonacci()->isValid(1); // true
v::fibonacci()->isValid('34'); // true
v::fibonacci()->isValid(6); // false
```
