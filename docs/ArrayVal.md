# ArrayVal

- `v::arrayVal()`

Validates if the input is an array or if the input can be used as an array
(instance of `ArrayAccess`).

```php
v::arrayVal()->validate([]); // true
v::arrayVal()->validate(new ArrayObject); // true
```
