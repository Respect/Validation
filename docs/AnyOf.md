# AnyOf

- `AnyOf(Rule ...$rule)`

This is a group validator that acts as an OR operator.

```php
v::anyOf(
    v::intVal(),
    v::floatVal()
)->validate(15.5); // true
```

In the sample above, `IntVal()` doesn't validates, but `FloatVal()` validates,
so `AnyOf()` returns true.

`AnyOf()` returns true if at least one inner validator passes.
