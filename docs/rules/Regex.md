# Regex

- `Regex(string $regex)`

Validates whether the input matches a defined regular expression.

```php
use Respect\Validation\Validator as v;

v::regex('/[a-z]/')->validate('a'); // true
```

Message template for this validator includes `{{regex}}`.

## Categorization

- Strings
