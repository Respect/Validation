# TrueVal

- `TrueVal()`

Validates if a value is considered as `true`.

```php
v::trueVal()->isValid(true); // true
v::trueVal()->isValid(1); // true
v::trueVal()->isValid('1'); // true
v::trueVal()->isValid('true'); // true
v::trueVal()->isValid('on'); // true
v::trueVal()->isValid('yes'); // true
v::trueVal()->isValid('0.5'); // false
v::trueVal()->isValid('2'); // false
```
