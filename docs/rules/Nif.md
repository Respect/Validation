# NIF

- `Nif()`

Validates Spain's fiscal identification number ([NIF](https://es.wikipedia.org/wiki/N%C3%BAmero_de_identificaci%C3%B3n_fiscal)).

```php
v::nif()->isValid('49294492H'); // true
v::nif()->isValid('P6437358A'); // false
```
