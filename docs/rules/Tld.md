# Tld

- `Tld()`

Validates whether the input is a top-level domain.

```php
v::tld()->isValid('com'); // true
v::tld()->isValid('ly'); // true
v::tld()->isValid('org'); // true
v::tld()->isValid('COM'); // true
```

- Internet
