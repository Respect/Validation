# Control

- `Control()`
- `Control(string ...$additionalChars)`

Validates if all of the characters in the provided string, are control
characters.

```php
use Respect\Validation\Validator as v;

v::control()->validate("\n\r\t"); // true
```

## Categorization

- Strings
