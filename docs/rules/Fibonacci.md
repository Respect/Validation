# Fibonacci

- `Fibonacci()`

Validates whether the input follows the Fibonacci integer sequence.

```php
use Respect\Validation\Validator as v;

v::fibonacci()->validate(1); // true
v::fibonacci()->validate('34'); // true
v::fibonacci()->validate(6); // false
```
